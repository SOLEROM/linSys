# keys

## defaults
```
Global
======
ctrl+shift+P    => command window
ctrl+,          => open settings
ctrl+\          => toggle tree view


Ctrl+P          => search and go to file
Ctrl+B          => searches through open files

TreeView
=========
Alt+\           => go to tree view
  A       add
  M       move
  Delete  delete


markdown
========
ctrl+shipt+m    => show markdown
```

## lines


```
duplicate line  (shift+ctrl+D)
remove line    (shift+ctrl+K)
move lines     (ctrl+keyUP/DOWN)


```


## override

```
shift-escape    =>  choose project
```
